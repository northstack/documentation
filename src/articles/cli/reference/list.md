# list

Lists all commands available within a namespace for the NorthStack CLI.

## Usage

`northstack list [options] [--] [<namespace>]`

## Arguments

-   **namespace**  
    The namespace name

## Options

-   **--raw**  
    To output raw command list
-   **--format=FORMAT**  
    The output format (txt, xml, json, or md) [default: "txt"]
